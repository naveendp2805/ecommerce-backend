# 🛒 Ecommerce Backend (Spring Boot)

## 🚀 Features
- Product CRUD with image upload
- Category management
- Product filtering by category
- Pagination and sorting
- DTO-based architecture
- Global exception handling

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

## 📌 APIs
- /api/products
- /api/categories

## ▶️ Run Locally
1. Clone repo
2. Configure database
3. Run Spring Boot app
